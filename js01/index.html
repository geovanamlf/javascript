Principais tipos primitivos de dados

  • Number: aceita todo tipo de numeros.
    1. infinity;
    2. NaN - not a number.

  • String: cadeia de caracteres.

  • Boolean: operadores logicos: true or false.

  • null: representa um valor nulo ou "vazio"

  • undefined: variável no escopo global. O valor inicial de undefined é o valor primitivo undefined.

  • object
    1. Arrary: é um conjunto de valores ordenados que você o referencia com um nome e um índice.

  • function: retorna um descritor de propriedade para uma propriedade nomeada em um objeto.
